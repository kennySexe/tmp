#### 가격을 입력해 크롤링 기술과 AI를 결합한 지능형 맛집 검색 서비스

##  주요 기능

**AI 대화형 검색**: `Deepseek AI 모델을 활용한 자연어 처리`

**실시간 지도 연동**: `Kakao Maps API 기반 위치 서비스`

**빅데이터 크롤링**: `Python Scrapy 기반 실시간 데이터 수집`

## 🛠️ 기술 스택

* Frontend
  * Next.js 15
  * React 18
  * TypeScript 5
  * Tailwind CSS 3.4
  * shadcn/ui

* AI & Backend
  * Deepseek AI
  * Python 3.11 (크롤링)

* APIs & Services
  * Kakao Maps API
  * Kakao Places API
  * Web Geolocation API
